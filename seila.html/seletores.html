CSS

CSS, ou Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem de estilo utilizada para definir a apresentação de documentos HTML (ou XML). Em outras palavras, o CSS é usado para controlar o layout, as cores, as fontes e outros aspectos visuais de uma página da web.

o que é um seletor

Um seletor em CSS é uma parte da regra CSS que define qual elemento HTML será estilizado. Os seletores são usados para direcionar os elementos aos quais se aplicarão as propriedades de estilo definidas em uma regra CSS.

tipos de seletores

Na área de desenvolvimento web, os seletores são usados em linguagens como CSS (Cascading Style Sheets) para direcionar elementos específicos em uma página HTML e aplicar estilos a eles.

propiedades, valores e regras CSS
